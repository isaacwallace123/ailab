# Kubernetes

Future Kubernetes manifests, Helm values, or GitOps app definitions for AI lab services.

Use this directory when an AI service benefits from:

- declarative deployment
- service discovery
- ingress
- health checks
- rollouts
- secrets integration
- metrics and logging

Do not move workloads here just because Kubernetes exists. A single-node model runtime may be simpler as a VM-level service until the service surface grows.
